# Phase 99 - Loan App Harassment Planner

Adds a safety-focused user tool for loan app harassment, overcharging, fake legal threats, contact-list threats and privacy misuse.

## Added

- `/tools/loan-app-harassment-planner`
- `LoanAppHarassmentPlannerForm`
- `buildLoanAppHarassmentPlan`
- `/admin/loan-app-readiness`
- `/api/admin/loan-app-readiness`
- `npm run loan-app:readiness`
- `npm run phase99:audit`

## Safety scope

- Guidance only, no legal/financial outcome promise.
- Strong warnings for OTP, UPI PIN, CVV, passwords, screen-share access, ID/bank/contact/photo privacy.
- Routes immediate threats and privacy abuse to trusted people and official emergency/local authority/cybercrime channels.
