# Phase 98 - Insurance Claim Planner

Adds a practical insurance claim planner for vehicle, health, travel, device, appliance, life, crop and other claims.

## Added

- `/tools/insurance-claim-planner`
- `/admin/insurance-claim-readiness`
- `/api/admin/insurance-claim-readiness`
- `npm run insurance-claim:readiness`
- `npm run phase98:audit`

## Safety rules

- Guidance only, no payout guarantee.
- No OTP, UPI PIN, CVV, password or remote access requests.
- Redact policy, ID, address, medical and bank details before public sharing.
- Use official insurer/TPA channels only.
