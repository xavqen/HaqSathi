# Phase 28 - Multi-channel Submission Pack

Added a mobile-first send-ready pack for complaints and support follow-ups.

## Added

- `/tools/submission-pack`
- `/dashboard/submission-packs`
- `/admin/submission-packs`
- `/api/tools/submission-pack`
- `SubmissionPack` Prisma model
- Header scroll nav entry
- Mobile bottom bar `Send` shortcut
- Dashboard/admin navigation entries
- `npm run phase28:audit`

## Output includes

- Formal email subject/body
- WhatsApp support message
- Support chat short message
- Call opening script
- Escalation message
- Social-safe public support post
- Evidence attachment order
- Mobile action plan
- Safety warnings

## Goal

After complaint generation, the user can immediately send the right message on the right channel without confusion.
