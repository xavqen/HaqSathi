# Phase 31 — Advanced Practical Helpers

Added four high-use, practical tools that make HaqSathi AI more useful for real Indian household and agent workflows.

## Added

- Chargeback / payment dispute helper
- Proof quality scanner
- Service center route planner
- Family case switchboard
- Dashboard history pages
- Admin monitoring pages
- Prisma models for all four tools
- Mobile header and bottom action shortcut updates

## Run

```powershell
npm install
npm run db:generate
npm run db:push
npm run phase31:audit
npm run scan:full
npm run dev
```
